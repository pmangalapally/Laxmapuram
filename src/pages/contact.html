Help Us Improve This Website

This is an open-source project — everyone can contribute!

🌟 Open Source Village Website

This website is maintained by volunteers who care about Laxmapuram. It is hosted for free on GitHub Pages and built with pure HTML, CSS, and JavaScript — no complex tools required. Whether you can write, code, take photos, or just correct a fact — you can contribute!

⭐ View on GitHub

How to Contribute

  1. Fork the repository — Go to github.com/Laxmapuram/Laxmapuram and click the Fork button to create your own copy.
  2. Make your changes — Edit existing pages, add new content, fix errors, or add photos. All content is plain HTML or Markdown.
  3. Test locally — Open the HTML files in your browser to verify your changes look correct. No server needed!
  4. Submit a Pull Request — Push your changes and open a Pull Request with a clear description of what you added or changed.
  5. Await review — A volunteer maintainer will review your PR, suggest any edits, and merge it when approved.

✍️ Content Writers

Help write or improve village history, festival descriptions, people profiles, and civic information. Knowledge of Telugu and the village is especially valuable. Even correcting a spelling mistake helps!

📸 Photographers

Submit photos of temples, festivals, landmarks, and everyday village life. See the Gallery page for photo submission guidelines. Good-quality images make the website come alive.

💻 Developers

Improve the HTML, CSS, or JavaScript. Fix accessibility issues, improve mobile responsiveness, add new features, or help translate content into Telugu. All skill levels welcome.

🙏 A Note of Thanks: Every contribution — big or small — honours the people, history, and culture of Laxmapuram. This website is a gift from the village community to future generations. If you are a resident, descendant, or admirer of Laxmapuram, your contribution is welcome and deeply appreciated.
